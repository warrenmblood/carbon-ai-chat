## Overview

Use `cds-aichat-code-snippet-card` to present generated code with syntax highlighting, copy affordances, and optional expand/collapse controls. The card container wraps a `cds-aichat-code-snippet` instance, you can show the code snippet without a card container by using `cds-aichat-code-snippet`, which has the same API.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-code-snippet-card highlight>
  import { summarize } from "./summarizer";
  export async function summarizeReply(prompt) {
    const result = await summarize(prompt);
    return result.text;
  }
</cds-aichat-code-snippet-card>
```
